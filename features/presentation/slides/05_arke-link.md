# `arke_link`: una tabella, tutte le relazioni

La colonna **`type`** ne cambia il significato:

| type                    | relazione           |
| ----------------------- | ------------------- |
| `parameter`             | Arke → i suoi campi |
| `group`                 | Arke → gruppo       |
| `permission`            | ruolo → Arke        |
| `category`, `author`, … | dominio             |
| `link`                  | generica            |

> Struttura, topologia, sicurezza — **tutto passa da qui.**
