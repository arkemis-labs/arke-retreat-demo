# Le quattro repo

- **`arke`** — il core: meta-modello, manager, query, hook
- **`arke-postgres`** — persistenza su PostgreSQL
- **`arke-auth`** — identità: utenti, autenticazione, permessi
- **`arke-server`** — layer HTTP (Phoenix)
